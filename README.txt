Create a popup and choose which page to display it on using a dropdown list.

== Description ==

Create a simple Popup, and put many different kinds of content in it: pictures, text, and even shortcode (see list of approved third party shortcodes)! The possibilities are endless. When someone visits the page that you want the popup to appear on, they will see the popup. Please note: <strong>not all shortcodes can execute properly in the popup</strong>. So far, only Gravity Forms shortcode have been integrated and tested. Place content in the popups at your own risk!

**Simple Popup Content Features:**

*   Create and manage as many popups as you want
#		Display Gravity Forms in a Popup
*		Display a YouTube video in a popup simply by pasting a YouTube link into the popup editor


**Popup Builder - 3rd party supported plugins:**

*   Gravity Forms


== Installation ==


<ul>
	<li>Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.</li>
	<li>Activate the plugin through the 'Plugins' screen in WordPress</li>
	<li>Add a Popup in the WordPress Dashboard area menu under Simple Popups -> Add New Popup Item</li>
	<li>Add content to the popup</li>
	<li>Choose which page to display the popup on from the dropdown list</li>
</ul>


== Frequently Asked Questions ==

= What can this plugin do? =

Create a popup and have it popup on any page you want by choosing a page from a dropdown list.
